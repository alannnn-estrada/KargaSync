# v0.3.0

## Auto-updates

- App now checks for new versions on startup across all platforms.
- **Windows**: downloads and installs updates silently in the background — a "Restart to install" prompt appears when ready.
- **macOS / Linux**: a notification banner links directly to the GitHub release page when a new version is available.

## Linux package repositories

- Official **apt** repository for Debian and Ubuntu — add once, get updates via `apt upgrade`.
- Official **yum/dnf** repository for Fedora and RHEL — add once, get updates via `dnf upgrade`.
- All installers also available for manual download on GitHub Releases.

## App icon

- Application icon now appears correctly in the taskbar (Windows), dock (macOS), and app launcher (Linux).
- High-resolution icon assets generated for all platforms.

## CI / CD

- GitHub Actions pipeline builds installers for Windows, macOS, and Linux on every release tag.
- Artifacts published automatically to GitHub Releases and the Linux package repositories.
